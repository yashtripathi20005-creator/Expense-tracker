# 💰 Expense Tracker with Charts

A comprehensive expense tracking application with interactive charts and visualizations built with Python, Streamlit, and Plotly.

## Features

- 📊 **Interactive Dashboard** - View all your expenses in one place
- 💰 **Add/Delete Expenses** - Manage your transactions easily
- 📈 **Visual Analytics** - Pie charts, line charts, and bar charts for spending analysis
- 🔍 **Filter & Search** - Filter by date range and search transactions
- 📱 **Responsive Design** - Works on desktop and mobile devices
- 🗂️ **SQLite Database** - Local storage for all your data
- 📊 **Summary Statistics** - Total expenses, averages, and top categories
